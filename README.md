# Vicsek Model with Hard Sphere Interactions and Density Dependent Velocity #

UzawaAlgorithmData_Kob_DensityVelocity.py FIle is used to  run the simulation and store the data without creating a video. You need to open the file and manually change the simulation parameters before running the python file.

Clusters.py Contains the functions to identify clusters using DBSCAN, fit ellipses using PCA and make videos from data

StatisticalAnalysis.py contains the functions calculate all the required statistics

Plots.ipynb uses all the presribed functions and generate all the required plots used to visualize the results from animations
